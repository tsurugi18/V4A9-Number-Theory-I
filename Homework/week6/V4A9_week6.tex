\par\textbf{(1)}\\
\par\textbf{(a)}
Taking the basis $\{e_1,\cdots,e_p\}\cup\{e_{p+1},\cdots,e_{p+q}\}$ given in the problem, we see for any $X\in U(p,q)$, 
\begin{equation*}
    h_A(Xe_i,Xe_j) = h_A(e_i,e_j) = \begin{cases}
        1 \quad i=j,\\
        0 \quad i\not=j.
    \end{cases}
\end{equation*}
From this we see that,
\begin{equation*}
    \mathbb{T}(A) = \diag(e^{i\theta_1}\otimes 1_A,\cdots,e^{i\theta_{p+q}}\otimes 1_A).
\end{equation*}
In particular, we have,
\begin{equation*}
    \mathbb{T}(\R) \cong (S^1)^{p+q}.
\end{equation*}
Therefore, $A_0=0$. We conclude $\mathbb{T}$ is not split.
\par\textbf{(b)}
We know that $U(n,n) \cong U(n,0)\times U(n,0)$. And for each $U(n)$, the borel subgroup is 
\begin{equation*}
    \mathbb{B}(A) = U(n,0)(A)\times U(0,n)\cap \{\text{upper triangular matrices}\}.
\end{equation*}
In particular $\mathbb{B}(\R) = \diag(e^{i\theta_1},\cdots,e^{i\theta_{2n}})$ since each column must be orthogonal to one another.
